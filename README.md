          +---------- DESKTOP AQUARIUM ----------+
A cozy, lofi, interactive aquarium simulation perfect for your 
desktop, complete with softbody physics and tank editing tools. 

          +------------ Installation ------------+
No installation is needed! Just download the source code from
https://github.com/Violet-Hackett/DesktopAquarium and run the
DesktopAquarium.exe file.
